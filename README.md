# Tide - Mobile Application

Tide is a modern, cross-platform mobile application designed with a focus on seamless user experience, clean architecture, and responsive UI design.

## 🛠️ Tech Stack
- **Framework:** [Flutter ]
- **Language:** [Dart ]
- **State Management:** Clean architecture for mobile state handling.

## 💡 Key Features
- **Cross-Platform Performance:** Optimized to run smoothly on both Android and iOS devices.
- **Modern UI/UX:** Built with native-like performance and responsive layouts for varying screen dimensions.
- **Modular Codebase:** Organized components ensuring scalability and ease of maintenance.

## ⚙️ How to Run and Build
1. Clone the repository: `git clone https://github.com/Manuel-Tyikusse/Tide.git`
2. Install dependencies: `[flutter pub get]`
3. Run the application: `[ios ou flutter run]`
